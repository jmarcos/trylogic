Template para a Demonstração no Coq I

O primeiro comando a ser feito no Assistente de Demonstração -- o Coq --, deve ser carregar a teoria que vamos utilizar para realizar as demonstrações em Dedução Natural.

Require Import ProofWeb.


Com este comando, carregamos todas as regras de inferência, que precisaremos para realizar demonstrações em Dedução Natural.


Em seguida, declaramos as variáveis que utilizaremos na demonstração:

Variables A B C : Prop.


Observe que declaramos A B C são do tipo Prop, ou seja, são variáveis proposicionais.


Na sequência, declaramos o que desejamos demonstrar, por exemplo:

Theorem example_01 : (A /\ B) -> (B /\ A).


Neste caso, é parte da comutatividade da conjunção, (α ∧ β) → (β ∧ α).
