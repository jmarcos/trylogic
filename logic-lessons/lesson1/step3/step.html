Template para a Demonstração no Coq II

Bom, agora que já vimos os templates de demonstração, para iniciar a demonstração é preciso o comando:

Proof.


No nosso exemplo anterior, iniciamos uma demonstração assim:

Require Import ProofWeb. Variables A B C : Prop. Theorem example_01 : (A /\ B) -> (B /\ A). Proof.


Agora, podemos inserir as regras de inferência adequadas até demonstrarmos o desejado.
