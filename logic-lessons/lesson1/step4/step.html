Template para a Demonstração no Coq

Vamos aplicar o Coq demonstrando o argumento: "α, α → β, β → (γ ∧ δ) ⊢ γ".
Nossa tarefa é produzir uma árvore de derivação, na qual as premissas pertencem às Folhas abertas, i.e. {α, α → β, β → (γ ∧ δ)} ⊂ Fp, e γ é o nó raiz .


Primeiro, carregamos a teoria com as regras de inferência:

Require Import ProofWeb.


Em seguida, declaramos as variáveis que utilizaremos na demonstração:

Variables A B C D: Prop.


Observe que A B C são do tipo Prop, ou seja, são variáveis proposicionais.


Na sequência, declaramos as etiquetas P1, P2 e P3 para as Premissas:

Hypothesis P1 : A.
Hypothesis P2: (A -> B).
Hypothesis P3: (B -> (C /\ D)).
E declaramos o que desejamos demonstrar :

Theorem exemple0 : C.

Para iniciar a demonstração é preciso o comando:

Proof.

Agora, sim podemos inserir as regras de inferência adequadas até demonstrarmos o desejado. Vemos nos próximos passos.
