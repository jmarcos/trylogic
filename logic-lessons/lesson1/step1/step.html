Sistema dedutivo Np no Coq

Nesta lição vamos apresentar um sistema dedutivo para a lógica proposicional: o sistema de dedução natural Np.


Um sistema dedutivo é um procedimento para se demonstrar que uma formula é consequência de um conjunto de premissas.


Nossa apresentação do sistema Np será realizada de forma computacional usando o assistente de demonstração chamado Coq.
