Sistema dedutivo Np no Coq

Nesta lição vamos apresentar um sistema dedutivo para a lógica proposicional: o sistema de dedução natural Np.


Um sistema dedutivo é um procedimento para determinar se uma dada fórmula é válida, ou não.


O procedimento envolve basicamente manipular os símbolos principais, de forma sintática, independetente de seus valores de verdade, ou seja, sem considerar qualquer noção semântica, ou as interpretações envolvidas.


Um sistema dedutivo é formado basicamente por um conjunto de regras de inferência.
